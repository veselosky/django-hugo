Welcome to the Django-Hugo Test Project

This is a simple Django project to test the integration with Hugo.

If you're seeing this message, presumably you have completed these first steps already:

  1. Run python manage.py migrate to set up the database.
  2. Run python manage.py runserver to start the development server.

To get started from here:

  1. Run python manage.py hugosync to load the default theme(s) into the database.
  2. Run python manage.py createsuperuser to create a user.
  3. Log in.
  4. Visit Your Sites to manage your Hugo sites.
